物化视图
	clickhouse作为一个从节点主动复制数据

SQL语句
查看库 SHOW CREATE DATABASE ck
查看表 SHOW CREATE tb
切换库 USE mysql_db
建库
	CREATE DATABASE [if not exists] ck ENGINE = atomic
	CREATE DATABASE [if not exists] mysql_db ENGINE = MySQL('ip:port','database','username','password')
建表
	CREATE TABLE default.tablename(`column` String)ENGINE = MergeTree ORDER BY column PARTITION BY toYYYYMM(date)
修改
	ALTER TABLE tablename UPDATE column = '' WHERE column = ''
删除表
	DROP TABLE tablename
创建枚举
	create table t_enum(a Enum8('hello' = 1, 'world' = 2)) ENGINE = TinyLog
去重
	OPTIMIZE TABLE tablename FINAL DEDUPLICATE BY column
导出
	clickhouse-client --query="SELECT * FROM tablename FORMAT CSV" > /tablename.csv
导入
	clickhouse-client --query="INSERT INTO tablename FORMAT CSV" < /tablename.csv
显示进行中的任务
	SHOW PROCESSLIST

注意
现代cpu不支持128为数字，Decimal128由软件模拟计算，所以运算速度不如Decimal64。
